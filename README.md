# TimeTable Manager

대학교 시간표를 자동으로 생성해주는 데스크톱 애플리케이션입니다.  
수강 희망 과목과 조건(목표 학점, 공강 요일, 최대 공강 교시)을 설정하면 가능한 시간표 조합을 자동으로 찾아줍니다.

---

## 주요 기능

- **시간표 자동 생성** — 목표 학점, 공강 요일, 최대 공강 교시 조건에 맞는 조합을 최대 50개 생성
- **분반 지원** — 같은 과목의 여러 분반 중 하나를 자동 선택
- **분리수업 지원** — 주 2회 수업처럼 여러 시간대가 세트로 묶인 과목 처리
- **PNP 과목 자동 포함** — Pass/Non-Pass 과목은 학점 계산 제외, 충돌 없으면 자동 편성
- **고정 과목** — 반드시 포함할 과목을 고정하여 조합 생성
- **이미지 내보내기** — 완성된 시간표를 PNG/JPG 파일로 저장
- **교시 시간 설정** — 학교마다 다른 교시 시작 시간을 직접 입력
- **저장/불러오기** — 과목 목록을 JSON 파일로 저장하고 재사용

## 설치 및 실행

### 요구 사항

- Python 3.10 이상
- PySide6

### 설치

```bash

git clone https://github.com/yourname/timetable-manager.git
cd timetable-manager

# 가상환경 생성 (권장)
```

## 사용 방법

### 1. 과목 입력

| 항목 | 설명 |
|------|------|
| 구분 | 전공 / 교양 / 기타 / PNP |
| 강의명 | 그룹명으로 사용됨. **같은 이름 = 같은 과목** |
| 분반 | 목록 표시용 레이블 (선택 사항) |
| 요일 / 시작·종료 교시 | 수업 시간대 |
| 학점 | 해당 과목의 이수 학점 |
| 분리수업 | 주 2회처럼 여러 시간대가 세트인 경우 체크 |

### 2. 그룹 개념 이해

같은 **강의명**을 가진 과목들은 하나의 **그룹**으로 묶입니다.

- **분반** (분리수업 미체크): 같은 이름으로 여러 시간대를 등록하면 시간표 생성 시 **하나만 선택**됩니다.  
  예) `영어회화 월3교시`, `영어회화 화5교시` → 둘 중 하나 선택

- **분리수업** (분리수업 체크): 같은 이름으로 등록된 슬롯이 **전부 함께 선택**됩니다.  
  예) `대학수학 월1-2교시` + `대학수학 목1-2교시` → 항상 세트로 포함

### 3. 고정 과목

강의 목록의 **고정** 체크박스를 활성화하면 해당 과목이 모든 시간표 조합에 반드시 포함됩니다.

### 4. 생성 조건 설정

| 항목 | 설명 |
|------|------|
| 목표 학점 | 완성 시간표의 총 이수 학점 (PNP 제외) |
| 최대 공강 교시 | 같은 날 수업과 수업 사이 허용되는 최대 빈 교시 수 |
| 공강 요일 | 수업을 배치하지 않을 요일 |

### 5. 결과 확인 및 저장

생성된 시간표는 오른쪽 탭에 번호 순으로 표시됩니다. 각 탭의 **이미지로 저장** 버튼을 누르면 PNG/JPG로 내보낼 수 있습니다.

## 알고리즘 개요

`logic.py`의 `ScheduleGenerator`는 다음 방식으로 동작합니다.

1. **고정 과목 사전 검증** — 충돌, 공강 요일 침범, 학점 초과 여부 확인
2. **그룹 옵션 구성** — 각 과목 그룹을 "분반 옵션들" 또는 "분리수업 세트"로 변환
3. **DFS + 접미 누적합 가지치기** — 학점 내림차순 정렬 후 남은 학점 합이 필요 학점보다 부족하면 탐색 중단
4. **PNP 자동 삽입** — 조합 확정 후 충돌·공강·공강 교시 조건을 만족하는 PNP 과목 자동 추가
5. **최대 50개 결과 반환**
